The classes of this package are used to retrieve content from the Web. Classes from the Semantic Web Client library were adapted to the Sesame framework and to perform retrieval using the Apache Commons HttpClient. A {@link CacheController} manages access to cached data. A {@link SpongerProvider} allows to utilize the Sponger functionalities of a local Virtuoso Server installation, which allow RDF extraction from a multitude of formats including GRDDL, RDFa and Microformats.
