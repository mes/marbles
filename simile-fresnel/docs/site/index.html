Fresnel

What is this?

Fresnel is a community developed ontology for displaying RDF; work in SIMILE also includes demonstration code for a Java-based, Fresnel-aware application. The home for Fresnel work has been relocated to http://www.w3.org/2005/04/fresnel-info/.

Meanwhile, SIMILE-specific Fresnel work continues in our wiki.
