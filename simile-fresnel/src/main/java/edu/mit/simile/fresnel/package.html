Implementation of the Fresnel core vocabulary, also includes Fresnel extension for facets and portions of the Fresnel extended vocabulary. The following is a minimal example of using the API.

Usage (key portions are in bold):

Graph confGraph, ontoGraph, dataGraph;

/* Here, read data into their respective graphs:
 * confGraph : Fresnel configuration
 * ontoGraph : Vocabularies used in data
 * dataGraph : RDF data
 */

try {
    Configuration conf = new Configuration(confGraph, ontoGraph);
    if (conf.hasWarnings()) System.err.println(conf.getWarningsString());

    Selection selected = conf.select(dataGraph);
    selected = conf.format(dataGraph, selected);
    Document out = selected.render();
    OutputFormat format = new OutputFormat(out); 

    // as a String
    StringWriter stringOut = new StringWriter();
    XMLSerializer serial = new XMLSerializer(stringOut, format);
    serial.setNamespaces(true);
    serial.serialize(out);

    // Display the XML
    System.out.println(stringOut.toString());
} catch (Exception e) {
    e.printStackTrace();
}
